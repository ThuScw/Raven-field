Ravenfield AI Threat Assessment Mod - Installation Guide
==========================================================

QUICK START (Method 1 - Recommended)
====================================
1. Make sure AIEnhancement.dll is in: Ravenfield_Data\Managed\
   (It should already be there)

2. Double-click "Launch with AI Enhancement.bat" to start the game

3. The mod will auto-load via Unity's [RuntimeInitializeOnLoadMethod]

4. Check the game log for [AIEnhancement] messages to confirm it's working


ALTERNATIVE METHODS
===================

Method 2: BepInEx (When Available)
----------------------------------
If you have BepInEx installed:
1. Copy AIEnhancement.dll to: BepInEx\plugins\
2. Launch game normally

Method 3: Manual Scene Edit (Advanced)
--------------------------------------
1. Copy AIEnhancement.dll to: Ravenfield_Data\Managed\
2. Use Unity or dnSpy to edit the main scene
3. Add GameObject with AIEnhancementInitializer component


VERIFICATION
============
To verify the mod is working:
1. Launch the game
2. Start a match
3. Open the log file: %USERPROFILE%\AppData\LocalLow\SteelRaven7\Ravenfield\output_log.txt
4. Search for "[AIEnhancement]" - you should see:
   - "[AIEnhancement] Threat Assessment System v1.0 Initializing..."
   - "[AIEnhancement] System active!"


UNINSTALL
=========
Simply delete AIEnhancement.dll from Ravenfield_Data\Managed\


FILES INCLUDED
==============
- Ravenfield_Data\Managed\AIEnhancement.dll (Main mod)
- Ravenfield_Data\Managed\Assembly-CSharp.dll.backup (Original game backup)
- Launch with AI Enhancement.bat (Convenient launcher)
- AIEnhancement\ (Source code and tools)


TROUBLESHOOTING
===============
Q: Mod doesn't load?
A: Make sure AIEnhancement.dll is in the Managed folder and not blocked by Windows.
   Right-click the DLL, select Properties, and click "Unblock" if present.

Q: Game crashes?
A: Delete AIEnhancement.dll and restore Assembly-CSharp.dll from backup if modified.

Q: AI behavior seems unchanged?
A: The mod only affects target selection. AI may still behave similarly in some situations.
   Look for AI switching targets more intelligently (e.g., prioritizing tanks over infantry).
