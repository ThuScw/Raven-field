# Ravenfield AI Enhancement V8

## 概述

V8 是一个全面的 AI 增强 MOD，实现了**单兵战术、小队协同、载具作战、随机战术**等全方位的AI行为增强。V8 在 V7 基础上进行了大规模重构，修复了多个逻辑缺陷，新增了精度系统、听声辩位系统、行动优先级仲裁等核心机制。

### 核心设计原则

**增强而非替代**

V8 严格遵循原版游戏的底层逻辑，只在策略和战术层面进行增强，不改变游戏的核心行为模式。

**优先级仲裁**

V8 引入行动优先级系统（ActionPriority），确保多个子系统不会同时争夺对同一个AI单位的控制权，高优先级行为自动覆盖低优先级行为。

## 安装方法

1. 确保游戏已关闭
2. 运行 `Injector.exe` 注入（如果尚未注入）
3. 直接双击 `Ravenfield.exe` 启动游戏
4. MOD 会自动加载

### 手动安装

如果注入器无法运行：

1. 将 `AIEnhancement.dll` 复制到 `Ravenfield_Data\Managed\`
2. 将 `Assembly-CSharp-patched.dll` 重命名为 `Assembly-CSharp.dll` 并替换原文件

### 卸载方法

复制 `Assembly-CSharp.dll.backup` 到 `Ravenfield_Data/Managed/Assembly-CSharp.dll`

## 技术细节

### 行动优先级系统

V8 引入 ActionPriority 枚举，防止多个子系统同时控制同一AI：

| 优先级 | 值 | 用途 |
|--------|---|------|
| Critical | 5 | 避让坦克路线等生存行为 |
| High | 4 | 掩体、协同等战术行为 |
| Medium | 3 | 听声辩位、目标选择 |
| Low | 2 | 巡逻、移动 |
| Minimum | 1 | 最低优先级行为 |
| None | 0 | 未被任何系统控制 |

每个更新周期开始时重置优先级状态，高优先级行为自动覆盖低优先级行为。

### 反射调用

V8 通过 AIUtils 共享工具类统一管理反射调用：

```csharp
AIUtils.SetTarget(ai, target, ActionPriority.High);
AIUtils.TryGoto(ai, position, ActionPriority.Medium);
```

### 更新频率

| 子系统 | 频率 | 说明 |
|--------|------|------|
| 精度系统 | 启动时 | 修改AI参数 |
| 掩体系统 | 0.3秒 | 快速响应压制状态 |
| 目标选择 | 0.3秒 | 快速响应战场变化 |
| 听声辩位 | 0.3秒 | 快速响应枪声 |
| 武器策略 | 1.0秒 | 慢速更新武器状态 |
| 小队系统 | 1.0秒 | 管理小队编制 |
| 协同系统 | 1.0秒 | 步坦/步机协同 |
| 载具系统 | 1.0秒 | 载具上车决策 |
| 随机战术 | 3.0秒 | 战术级随机行为 |
| 数据清理 | 30秒 | 清理无效Actor数据 |

### 数据清理

V8 每30秒自动清理已死亡/移除的Actor数据，防止静态字典内存泄漏。

### 线程安全

所有代码在 Unity 主线程运行，无需额外同步。

## 已知限制

1. 所有AI行为基于原版游戏逻辑，可能出现与原游戏略有不同的表现
2. 随机战术行为触发频率较低，需要长时间游戏才能观察到
3. 载具躲避障碍物功能依赖于Unity物理引擎的Raycast
